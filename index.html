Electrum


Confira mais detalhes :

Construi o front End de um e-commerce chamado Electrum, continuação do aprendizado do Framework SASS, focado no design patterns usados no dia-a-dia, para simular o máximo possível um projeto front end real, está dividido em sua estilização em módulos para facilitar manutenções futuras.

Módulo 1-base: nesse módulo fica todas as tags essencias que não possuem classes ou id's, nesse mesmo módulo também é adicionado o reset e variáveis Já no módulo layouts como o nome mesmo indica fica toda a estilização do site, contendo todas as sessões do site separadas, todas as classes e id's.

Assim , facilitando uma possível manutenção ou atualização do código, Já no terceiro e último módulo está o Modules, todos os componentes que são reutilizados ao decorrer do projeto e que aparecem ou se repetem, acabam ficando nesta pasta, assim facilita o reuso do mesmo.

Desafio Nacional


Confira mais detalhes :

Projeto desenvolvido para concorrer ao Desafio Nacional promovido pela Unicesumar, ao qual eu fiquei entre os 10 melhores votados no Brasil, o desafio em sí era uma forma de ajudar alguém da comunidade com um site p/ promover e aumentar as vendas, o projeto foi um site de marmitaria.

O que acabou facilitando o acesso ao cardápio, disponibilizando um contato direto para fazer pedidos via whatsApp e links paras redes sociais, Neste projeto trabalhei com back-end em php e seccionei em partes para facilitar a manutenção. Para selecionar e navegar entre o cardápio do site eu trabalhei com array dinâmico, para estilizar, CSS3 e fiz o máximo que pude na época para colocar todo o html em tags semânticas.

Por ser o primeiro contato com a linguagem Php, fiquei satisfeito em ter ficado ali nos 10 melhores projetos a nivel nacional, com certeza seria uma honra ter a conquista do primeiro lugar, mas sei que ainda preciso avançar e melhorar em muitos aspectos.
