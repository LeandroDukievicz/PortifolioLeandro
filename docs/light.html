Electrum


Confira mais detalhes :

Construi o front End de um e-commerce chamado Electrum, continuação do aprendizado do Framework SASS, focado no design patterns usados no dia-a-dia, para simular o máximo possível um projeto front end real, está dividido em sua estilização em módulos para facilitar manutenções futuras.

Módulo 1-base: nesse módulo fica todas as tags essencias que não possuem classes ou id's, nesse mesmo módulo também é adicionado o reset e variáveis Já no módulo layouts como o nome mesmo indica fica toda a estilização do site, contendo todas as sessões do site separadas, todas as classes e id's.

Assim , facilitando uma possível manutenção ou atualização do código, Já no terceiro e último módulo está o Modules, todos os componentes que são reutilizados ao decorrer do projeto e que aparecem ou se repetem, acabam ficando nesta pasta, assim facilita o reuso do mesmo.
